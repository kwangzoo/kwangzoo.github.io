# Kwangzoo Chung

This website is based on the open source templates designed and shared by the labs of [D. Allan Drummond](http://www.allanlab.org/aboutwebsite.html) and [Trevor Bedford](http://bedford.io/misc/about/). I downloaded source codes from their repository and modified our contents based on shared templates. I greatly thank for Allan and Trevor for allowing reuse of their codebase. 

The website is deployed using [GitHub Pages](https://kwangzoo.github.io) and the source code is available on [GitHub](https://github.com/kwangzoo/kwangzoo.github.io). Please feel free to reuse this code (making sure to cite the Bedford lab and Drummond lab as the original sources of the lab website template).
